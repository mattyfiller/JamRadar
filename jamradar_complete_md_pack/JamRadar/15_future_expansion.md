# 15 - Future Expansion

## Expansion Path

### Phase 1

Ski/snowboard events and rail jams.

### Phase 2

Indoor training facilities and gear deals.

### Phase 3

Skate events and gear swaps.

### Phase 4

Used gear marketplace.

### Phase 5

MTB, BMX, wake, and broader action-sport events.

## Future Feature Ideas

### AI Event Import Assistant

Organizers or users paste a link or upload an event poster. AI extracts:

- Title
- Date
- Time
- Location
- Cost
- Registration link
- Description
- Sport
- Event type

This would make event submission much faster.

### Event Poster Scanner

User uploads a screenshot from Instagram. App turns it into a structured event submission.

### Crew Mode

Users create small friend groups.

Features:

- Share events
- See who saved what
- Vote on events
- Plan weekend trips

### Weekend Send Planner

App recommends the best nearby events for the weekend based on:

- Distance
- Weather
- User interests
- Skill level
- Friend interest
- Cost

### Weather Integration

For outdoor mountain events:

- Snowfall
- Temperature
- Wind
- Road conditions, maybe later

### Resort/Festival Passes

Users can buy event passes or register through the app.

### Athlete Profiles

Optional future social layer:

- Rider profiles
- Event results
- Clips
- Sponsors
- Competition history

### Results and Recaps

After events, organizers can post:

- Winners
- Photos
- Videos
- Sponsor shoutouts
- Next event link

### Loyalty System

Users earn points for:

- Attending events
- Checking in
- Sharing events
- Buying from partners
- Volunteering

Rewards:

- Discounts
- Stickers
- Partner coupons
- Free entries

### Marketplace Advanced Tools

- Secure payments
- Shipping labels
- Seller ratings
- Price suggestions
- Trade offers
- Saved searches
- Gear alerts

## Long-Term Vision

JamRadar can become the default discovery layer for local action-sport culture.

Not just “what events exist?”

But:

- What is happening near me?
- What should I not miss?
- Where can I train?
- Where are the gear deals?
- Who is hosting the next jam?
- What should my crew do this weekend?
