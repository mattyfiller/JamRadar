# 08 - Monetization Strategy

## Best Monetization Order

Do not charge regular users immediately.

The best early monetization path is:

1. Free app for users
2. Paid tools for organizers, shops, facilities, and brands
3. Sponsored event listings
4. Gear deals and affiliate revenue
5. Used gear marketplace transaction fees
6. Premium user features later

## Revenue Stream 1: Organizer Subscriptions

Mountains, shops, brands, clubs, indoor facilities, and event organizers can pay for posting and promotion tools.

### Free Tier

- Basic profile
- Limited events per month or season
- Basic event posting
- No advanced analytics

### Basic Tier

Possible price: $19-$49/month

Includes:

- More events
- Event images
- Registration links
- Basic analytics
- Update notifications to saved users

### Pro Tier

Possible price: $99-$199/month

Includes:

- Unlimited events
- Featured placement credits
- Advanced analytics
- Follower notifications
- Sponsor logos
- Multi-event campaigns

### Enterprise/Resort Tier

Custom pricing.

For large mountains, event series, brands, and multi-location facilities.

## Revenue Stream 2: Featured Events

Organizers can pay to promote events.

Examples:

- Featured in local feed
- Featured on map
- Featured in weekly digest
- Featured in category feed

Possible pricing:

- $25-$100 for local small events
- $100-$500 for regional campaigns
- $500+ for brands or larger resorts

## Revenue Stream 3: Brand Sponsorships

Brands can sponsor:

- Rail Jam category
- Weekly event digest
- Map view
- Gear deal section
- Indoor training category
- Best Trick event series

Example:

“Rail Jam alerts presented by Brand X.”

## Revenue Stream 4: Gear Affiliate Deals

JamRadar earns commission from gear purchases.

Sources:

- Online retailers
- Local shops
- Brand partner links
- Seasonal buying guides
- Event-related product links

## Revenue Stream 5: Local Shop Promotions

Shops can pay to post:

- Sales
- Demo days
- Tuning clinics
- Wax nights
- Team events
- Gear swaps

This is a very realistic early revenue source because local shops always need community visibility.

## Revenue Stream 6: Used Gear Marketplace Fee

Once the marketplace is mature:

- Take 5%-10% of transactions
- Charge listing boosts
- Charge shop accounts for bulk listings
- Offer verified seller tools

## Revenue Stream 7: Premium User Subscription

This should come later.

Possible features:

- Advanced alerts
- Multi-region tracking
- Calendar sync
- Gear price alerts
- Deal watchlist
- Ad-free app
- Trip planning tools
- Crew planning tools

Possible price:

- $3-$7/month
- $20-$40/year

## Revenue Stream 8: Ticketing and Registration

Eventually, JamRadar can process registrations directly.

Revenue:

- Small service fee
- Percentage of registration
- Organizer payment processing fee

This is powerful but should not be an MVP feature because payments, waivers, refunds, and liability make it more complex.

## Recommended Early Monetization Strategy

Start with:

1. Free user app
2. Free organizer posting during launch
3. Paid featured event placements
4. Local shop deal promotions
5. Indoor facility promoted sessions
6. Affiliate gear links

Then add marketplace fees later.
