# 10 - MVP Scope

## MVP Goal

Build a simple app that helps users discover and get notified about ski/snowboard events, rail jams, indoor training sessions, and gear-related local events within a chosen radius.

## MVP Should Prove

1. Users want a centralized event feed.
2. Users value radius-based notifications.
3. Organizers are willing to post events.
4. Gear/shop integrations can generate revenue.
5. Indoor training facilities can help with off-season engagement.

## MVP User Features

- User account
- Location/radius setup
- Sport preferences
- Event type preferences
- Event feed
- Event details
- Save event
- Push notifications
- Basic map view
- Basic calendar view
- Gear deals tab

## MVP Event Types

- Rail jam
- Terrain park event
- Ski competition
- Snowboard competition
- Demo day
- Indoor training session
- Gear swap
- Skate jam, optional depending on region

## MVP Organizer Features

- Organizer account
- Create organization profile
- Create event
- Edit event
- Upload event image
- Add registration link
- Add booking link for indoor sessions
- View basic analytics

## MVP Admin Features

- Approve organizer accounts
- Approve user-submitted events
- Edit event details
- Remove spam
- Feature events manually

## MVP Gear Features

- Gear deal listings
- Local shop deal posts
- Affiliate link support
- Sponsored deal label

## MVP Marketplace Decision

Do not build the full used gear marketplace in MVP.

Instead, include:

- Gear deals
- Gear swap events
- Local shop used gear posts

Build the full marketplace after the event app has active users.

## MVP Platforms

Recommended:

- Mobile-first web app first, or
- React Native / Expo app if push notifications are essential from day one

A mobile-first web app is faster to test.
A native-style app is better for notifications.

## MVP Success Metrics

User metrics:

- Number of signups
- Notification opt-in rate
- Events saved per user
- Event detail views
- Registration clicks
- Weekly active users

Organizer metrics:

- Number of organizers onboarded
- Events posted
- Repeat event posting
- Featured placement interest
- Gear deal posts

Revenue metrics:

- Sponsored event sales
- Shop deal sales
- Affiliate clicks
- Indoor facility interest
