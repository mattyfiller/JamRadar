# 05 - Feature List

## Core User Features

### Account and Profile

- Email login
- Apple/Google login
- Profile photo
- Sports interests
- Home area
- Travel radius
- Favorite mountains/facilities
- Skill level
- Notification settings

### Event Feed

- Personalized event feed
- Radius-based filtering
- Sport filtering
- Event type filtering
- Skill-level filtering
- Free/paid filtering
- Indoor/outdoor filtering
- Date filtering
- Saved events
- Trending events

### Event Detail Page

Each event should include:

- Event title
- Date and time
- Location
- Map
- Description
- Event type
- Sport
- Skill level
- Cost
- Registration link
- Registration deadline
- Age groups
- Categories
- Prize info
- Sponsor info
- Organizer profile
- Contact info
- Weather info, later phase
- Change log, later phase

### Notifications

Notification types:

- New event near you
- New rail jam near you
- Registration closing soon
- Saved event reminder
- Event changed
- Event cancelled
- New indoor session added
- Gear deal matching your interests
- Weekly event digest

### Map View

- Event pins
- Venue pins
- Indoor facility pins
- Shop pins
- Filtered map layers
- Distance estimates

### Calendar View

- Upcoming saved events
- Nearby events by date
- Add to phone calendar
- Weekly planning view

### Search

Search by:

- Event name
- Resort
- City
- Sport
- Event type
- Shop
- Brand
- Indoor facility
