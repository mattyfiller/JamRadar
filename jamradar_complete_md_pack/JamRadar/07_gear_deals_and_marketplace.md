# 07 - Gear Deals and Marketplace

## Should JamRadar Include Gear Deals?

Yes, but gear deals should not overwhelm the main event experience.

Gear deals are a smart addition because the same users who attend rail jams, skate events, and mountain events also buy:

- Skis
- Snowboards
- Boots
- Bindings
- Helmets
- Goggles
- Jackets
- Pants
- Gloves
- Wax
- Tuning tools
- Skateboards
- Trucks
- Wheels
- Pads
- MTB gear

The key is to make gear deals relevant, not spammy.

## Best Way to Add Gear Deals

Gear deals should be personalized by:

- Sport
- Size
- Location
- Preferred brands
- New vs used
- Budget
- Season
- Event context

Example:

If a user saves a rail jam, they might see:

- Wax deal
- Helmet discount
- Local shop tuning coupon
- Board/ski sale
- Gloves under $50

## Gear Deal Categories

- Local shop deals
- Brand deals
- Event sponsor deals
- Seasonal sales
- Demo gear sales
- Clearance gear
- Tune/wax coupons
- Rental discounts
- Helmet/goggle deals
- Skate gear
- Used gear

## Monetization From Gear Deals

### 1. Affiliate Commission

JamRadar earns a percentage when users buy through links.

Good for:

- Online retailers
- Gear brands
- Large shops

### 2. Sponsored Deal Placement

Shops/brands pay to promote deals in a region.

Example:

- “Featured deal in Ontario snowboarding feed”
- “Sponsor the Rail Jam category this week”

### 3. Local Shop Subscription

Shops pay monthly to list:

- Deals
- Events
- Gear swaps
- Demo days
- Tuning clinics

### 4. Coupon Tracking

JamRadar gives shops unique codes.

Example:

- JAMRADAR10
- SENDMAP15

This helps prove value.

## Should JamRadar Include Used Gear Buy/Sell?

Yes, but it should probably be Phase 2 or Phase 3, not the first MVP.

A used gear marketplace is valuable because action-sport gear is expensive and people constantly buy, sell, upgrade, outgrow, or change setups.

This could become a strong revenue stream if handled safely.

## Used Gear Marketplace Concept

Users can list:

- Used snowboards
- Used skis
- Boots
- Bindings
- Helmets, with safety warnings
- Goggles
- Outerwear
- Skateboards
- Wheels/trucks
- MTB gear later
- Tuning tools

## Marketplace Filters

Users should filter by:

- Sport
- Category
- Size
- Length
- Boot size
- Binding size
- Brand
- Price
- Condition
- Distance
- Local pickup/shipping

## Marketplace Revenue Model

### Option 1: Transaction Fee

JamRadar takes a percentage when users buy through the app.

Possible fees:

- 5% seller fee
- 8% seller fee
- 10% total platform fee
- Flat $2-$5 fee for small items

### Option 2: Paid Boosts

Sellers can pay to boost listings.

Example:

- $3 to boost for 3 days
- $5 to boost for 7 days

### Option 3: Shop Used Gear Listings

Local shops can list used/demo gear with a paid account.

### Option 4: Verified Seller Badge

Shops or trusted sellers can pay for verified status.

## Marketplace Safety Concerns

This area needs care.

Important issues:

- Scams
- Fake listings
- Stolen gear
- Unsafe helmets
- Misrepresented condition
- Payment disputes
- Shipping disputes

## Safety Features

- User verification
- Report listing button
- Listing review for high-value items
- Seller ratings
- Buyer ratings
- Secure payments, later phase
- Local pickup safety tips
- Required condition photos
- Serial number field for bikes/skis/boards where useful
- No sale of unsafe/damaged protective equipment without warning

## Important Helmet Note

Used helmets are tricky because damage may not be visible. The app should include a safety notice for used helmets and possibly restrict helmet sales or require clear disclaimers.

## Best MVP Approach for Gear

For the MVP, do this:

- Add gear deal section
- Allow shops to post deals
- Use affiliate links
- Add event-related sponsor deals

Do not build full used gear payments at first.

For Phase 2:

- Add simple local used gear listings
- Allow buyer/seller messaging
- No in-app payments yet

For Phase 3:

- Add secure payments
- Take a percentage
- Add seller ratings
- Add shipping

## Why Gear Makes Strategic Sense

Gear gives JamRadar a path to revenue outside events.

Events bring users in.
Gear creates ongoing monetization.
Marketplace creates network effects.

This combination is stronger than events alone.
